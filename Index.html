CSS Basics

CSS (Cascading Style Sheets) is the code that styles web content.

As the title suggests, this article will give you a basic understanding of CSS and its functions.

So what is CSS?

CSS is a style sheet language that is used to selectively style HTML elements. For example, you can use CSS to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features. Unlike HTML, CSS is not a mark up language.

Anatomy of a CSS ruleset

The individual parts of a CSS ruleset are as follows;

  1. Selector: HTML element at the start of the ruleset which defines the element to be styled. The most common types are the element selector, id selector, class selector, attribute selector and the pseudo attribute selector.
  2. Declaration: a single rule which specifies which of the element's properties you want to style.
  3. Properties: ways in which you can style an HTML element.
  4. Property value: chooses one out of many possible appearences for a given property.

CSS Layout

This is mostly based on the box model with each box taking up space on a page. This page includes properties like;

  • padding - space around content
  • border - solid line just outside the content
  • margin - space around outside of the border
  • width - of an element
  • background-color - color behind an element's content and padding
  • color - color of an element's content (usually text)
  • text-shadow - sets a drop shadow on the text inside an element
  • display - sets the display mode of an element

Conclusion

While HTML is used to define the structure and semantics of your content, CSS is used to style it and lay it out.
